# kotlin-lab6
##  Description
Projet Kotlin pour le TP/Lab 6.
Ce TP introduit la programmation orientée objet en Kotlin à travers la création d’une classe Player et d’une classe Scores.
On simule plusieurs niveaux de jeu pour deux joueurs, on calcule leurs scores totaux et on détermine le gagnant.
